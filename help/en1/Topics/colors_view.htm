Color View

Use Color View to specify both the text color (fore color) and background color (back color). You can specify any of the following colors for both text and background:

  • Black
  • Red
  • Green
  • Yellow
  • Blue
  • White

The default text color is black, and the default background color is white.

You can also use Color View to enter or change the text in the text box.
