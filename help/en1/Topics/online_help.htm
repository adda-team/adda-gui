Online Help

Typefacer has the following options for using online help:

  • Embedded help, in which help is displayed in the application interface. As you move from screen to screen, the online help refreshes to display help for the current screen. You can show or hide embedded help using the Help menu.
  • Field-level help, in which you can request help for a specific control on the screen. When you click the button and then click a control, Typefacer displays help for that specific control.
  • Standard help, in which you can display the online help and use a table of contents, index, or word search index to find topics. You can access standard help from the Help menu.

By default, Typefacer opens with embedded help displayed.
