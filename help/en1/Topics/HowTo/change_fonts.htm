How to Change Fonts

Use the following steps to change fonts:

  1. If it is not already active, select Typeface View from the View menu.
  2. Ensure that you have text typed in the Text box.
  3. Click the Font dropdown box, and select a font from the list.
  4. Click the Show button to display the style changes.
