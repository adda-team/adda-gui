How to Apply Text Styles

Use the following steps to apply text styles:

  1. If it is not already active, select Typeface View from the View menu.
  2. Ensure that you have text typed in the Text box.
  3. To apply the bold style, select the Bold checkbox. To remove the bold style, clear the Bold checkbox.
  4. To apply italics, select the Italics checkbox. To remove the italics, clear the Italics checkbox.
  5. Click the Show button to display the style changes.
