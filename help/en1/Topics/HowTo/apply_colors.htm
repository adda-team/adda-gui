How to Apply Colors

Use the following steps to apply colors:

  1. If it is not already active, select Color View from the View menu.
  2. Ensure that you have text typed in the Text box.
  3. To change text color, click the Fore Color dropdown box, and select a color from the list.
  4. To change background color, click the Back Color dropdown box, and select a color from the list.
  5. Click the Show button to display the color changes.
