Internal fields and dipole polarizations

ADDA can save internal electric fields Ei and/or dipole polarizations Pi at each dipole (§10) to files IntField and DipPol respectively (§C.9), using command line options

-store_int_field

-store_dip_pol

These options allow one to study in details the accuracy of the DDA (see e.g. [17]) or the physics behind it. To save internal fields for future use by ADDA consider using checkpoints of type “always” (§12.5).
