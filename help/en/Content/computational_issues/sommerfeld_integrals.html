ADDA calculates Sommerfeld integrals using slightly modified routine from NEC2C,[84] which is a Fortran-to-C translation of [52]. This routine contains internal error control, and has been tested to produce consistent (within 0.1%) results in comparison with other codes [84,85]. Still, it should be regarded as a black box. While the accuracy is expected to be sufficient for most applications, it is hard to confidently put any numbers on it.

To accelerate computations ADDA first computes a table of Sommerfeld integrals for all unique values of ρ and Z required for dipoles on the computational grid, which takes time comparable to 200 iterations of the iterative solver[85]. Moreover, such table requires additional memory (ยง5).
